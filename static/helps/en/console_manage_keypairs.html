Manage Keypairs

This screen allows you to view a list of your keypairs, create new keypairs, and delete keypairs. You can page through the list of keypairs by clicking the navigation buttons at the bottom of the screen.

Sorting the Keypair List

Click on the Name column header to toggle sorting the list of keypairs in ascending or descending order.

Searching the Keypair List

Type some search text into the Search text box to filter the list of keypairs.

Creating Keypairs

Click the Create Keypair button. The Create Keypair dialog box will appear.

Deleting Keypairs

  1. Click the checkbox next to the keypair.
  2. Right-click in the keypair list and select Delete from the context menu. The Delete Keypair dialog box will appear.
