Navigate the Dashboard

The dashboard is your starting point for using the Eucalyptus console. From the Dashboard, you can access instances, storage items (volumes, snapshots, and buckets), and networking and security objects (keypairs, security groups, and IP addresses).

Basic Dashboard Navigation

You can navigate to specfic resource management dialogs in two ways: using the drop-down navigation menu or clicking directly on a resource type in the Dashboard screen.

  1. To use the drop-down navigation menu, click on the Manage Resources drop-down menu at the top of the dashboard. A navigation menu appears that allows you to select a resource to manage.
  2. You can also navigate to a resource management screen by clicking directly on a resource icon in the main dashboard window.

Instances

The Instances section allows you to see how many instances are running and access the Manage Instances screen.

  1. Simply click anywhere in the Instances section of the dashboard to display the Manage Instances screen.
  2. You can filter the Instances display by availability zone by selecting an availability zone in the drop-down list box.
  3. You can launch a new instance by clicking the Launch instance link to display the Create instance dialog box.

Storage

The Storage section allows you to see at a glance how many storage objects are running and access the various storage object management screens.

  1. To access the Manage Volumes screen, click the Volumes icon.
  2. To access the Manage Snapshots screen, click the Snapshots icon.
  3. To access the Manage Buckets screen, click the Buckets icon.

Network and Security

The Network and Security section allows you to see at a glance how the number of security groups, key pairs, and IP addresses are in your Eucalyptus cloud, and to navigate to management screens for each type of object.

  1. To access the Manage Security Groups screen, click the Security Groups icon.
  2. To access the Manage Key Pairs screen, click the Key Pairs icon.
  3. To access the Manage IP Addresses screen, click the IP Addresses icon.
