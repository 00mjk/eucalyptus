Manage Instances

This screen allows you to view a list of your instances, create new instances, and delete instances. You can page through the list of instances by clicking the navigation buttons at the bottom of the screen.

Sorting the Instances List

Click on any column header to toggle sorting the list of instances in ascending or descending order by the selected column.

Searching the Instance List

Type some search text into the Search text box to filter the list of instances.

Launch an Instance

Click the Launch new instance button. The Launch Instance dialog box will appear.
